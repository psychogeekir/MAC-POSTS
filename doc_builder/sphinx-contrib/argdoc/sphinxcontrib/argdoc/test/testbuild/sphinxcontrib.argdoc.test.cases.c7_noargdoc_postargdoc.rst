Use of 'noargdoc' function decorator

The :func:`~sphinxcontrib.argdoc.ext.noargdoc` function decorator tells
:obj:`sphinxcontrib.argdoc` to ignore a command-line script, leaving
its ``:automodule:`` output unchanged.
